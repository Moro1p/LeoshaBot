# 🎮 LeoshaBot

Набор инструментов для стриминга на Twitch.

Включает Twitch-бота для отображения текущего трека из Яндекс Музыки и экспериментальную систему Vtuber-аватара, реагирующего на голос с микрофона.

</div>

---

## Текущие возможности

### 🎵 Twitch Music Bot

Показывает текущий трек из Яндекс Музыки по командам чата.

Поддерживаемые команды:

* `!np`
* `!song`
* `!music`
* `!песня`
* `!музыка`

Пример ответа:

```text
Linkin Park - Numb: clck.ru/xxxxx
```

---

### 🎙️ Audio Reactive Vtuber (Experimental)

Экспериментальная система анимированного PNG-аватара.

Возможности:

* отслеживание громкости микрофона в реальном времени;
* автоматическая анимация рта;
* плавное переключение кадров;
* эффект тряски при повышении громкости;
* прозрачное окно поверх всех окон Windows;
* масштабирование аватара клавишами `+` и `-`;
* перетаскивание мышью;
* управление персонажами через встроенный интерфейс;
* хранение наборов персонажей в папке `Sprites`.

Структура спрайтов:

```text
Sprites/
│
├── Character1/
│   ├── 0.png
│   ├── 1.png
│   ├── 2.png
│   └── 3.png
│
└── Character2/
    ├── 0.png
    ├── 1.png
    ├── 2.png
    └── 3.png
```

Где:

* `0.png` — закрытый рот;
* последние кадры — максимально открытый рот;
* кадры переключаются в зависимости от уровня громкости.
